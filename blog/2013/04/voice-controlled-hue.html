Voice & Weather Controlled Philips Hue Lights

Apr 27, 2013  •  #Philips Hue #Speaking 

Voice control

This is a very simple demonstration of using voice commands to control Philips Hue light bulbs. In this example, I’m using Chrome’s x-webkit-speech speech input and Dan Ray’s phpHue to interact with the Hue system.
